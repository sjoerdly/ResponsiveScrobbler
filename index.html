Responsivescrobbler

Download .zip Download .tar.gz View on GitHub

This is a hobby project to provide custom scrobbling to Last.fm for tracks played on physical media or other sources you can't scrobble automatically. This is not intended to mass-scrobble or misuse the last.fm systems in any way.
